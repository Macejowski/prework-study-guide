CSS

  • Enter your CSS notes here,
  • CSS has three levels of scope: Inline, Internal and External.
  • Inline CSS is coded directly within the HTML file within the tag it applies to.
  • Internal CSS is a stylesheet embedded in an HTML file and applies to all tagged elements on that page.
  • External CSS are separate files that an HTML file uses the link element to link.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside the element.
  • Colors can be defined semanticly or in hexdecimal
